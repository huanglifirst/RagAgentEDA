Bandgap设计

本教程将通过分析提供的 Bandgap Reference (bgref) 代码，向您展示如何利用 TED 框架进行类似电路的设计。我们将遵循从底层元件到顶层集成的顺序，解释关键模块的实现，并结合代码片段说明 TED 的使用方法和设计技巧。

由于提供的代码库包含多个部分（bgpref - 一个具体的实现，bgp - 不同拓扑的生成器，bgp_mir - 基于电流镜的优化版本），我们将重点关注 bgpref 部分的实现流程，因为它包含了较完整的从模块设计到版图集成的过程，然后简要介绍 bgp 生成器模式和 bgp_mir 的优化方法。

第一部分：bgpref - 一个具体的带隙基准实现

这部分代码 (src/bgpref/) 实现了一个功能相对完整的带隙基准电路，包含了核心电路以及 UVLO、温度保护、Trim 等外围功能。

1. 基础与辅助模块

设计始于基础单元和一些辅助模块。

  • 基础器件 (来自 ted_device): 代码中广泛使用了 TED 提供的基础器件模型，如：

    • NMOS, PMOS: MOS 晶体管。代码中通过 with setenv(nch="nch_v1", pch="pch_v1") 等方式指定具体使用的模型。
    • BJT: 双极结型晶体管，用于产生 VBE 和 PTAT 电流。
    • RpolySingle: Poly 电阻单元。
    • Capacitor: 电容器。
    • Pfuse: 可熔断保险丝，用于 Trim。
    • IProbe, VSource, ISource, Clock: 仿真激励和探针。
    • Standard Cells: Inv, Nor2, SMIT (施密特触发器)。

  • 辅助模块 (自定义或来自库):

    • ParallelResistor, SeriesResistor: 用于方便地创建串联或并联的电阻网络，底层调用 RpolySingle。
    • CurrentMirror: 电流镜模块，可能封装了匹配布局。
    • PairedMosfet: 差分对 MOS 管，可能包含交叉耦合等匹配技术。
    • ref_trim.py 中的 Latch: 一个简单的锁存器单元，用于存储 Trim 位。

  • 代码示例 (ref_trim.py/Latch):
    Python
    # tedhub/circuit/bgref/src/bgpref/ref_trim.py
    @module
    def Latch(A, tgp: Ctrl, tgn: Ctrl, Y: Output):
    # 两个反相器交叉耦合
    A >> (i1 := Inv()) >> Y
    Y >> (i2 := INV_TG()) % (tgp, tgn) >> A # INV_TG 可能是带传输门控制的反相器
    # --- 版图布局 ---
    i2.port("Y")[0].set_port() # 标记端口以便布线器连接
    i1.port("O")[0].set_port()
    final = row(i2, i1, align=MIDDLE) # 水平排列
    place() # 放置
    route(via=1, top_layer="M2", disable_priority=True) # 布线

  • TED 设计技巧:

    • 利用 ted_device 提供的丰富器件库。
    • 创建辅助模块 (如 ParallelResistor, Latch) 封装常用电路结构，提高代码复用性。
    • 在模块定义中使用 place(), route() 和布局函数 (row, col) 来描述版图。

2. 核心子模块

bgpref 电路被分解为多个功能子模块，每个模块都有对应的 _func (定义连接关系和子单元) 和主模块 (调用 _func 并进行版图布局)。

2.1 Vdda 分压模块 (vddadiv.py)

  • 功能: 对输入的 Vdda (模拟电源) 进行分压，可能用于产生内部参考电压或为 UVLO 提供阈值。

  • 实现:

    • vddadiv_func: 使用 ParallelResistor 和 SeriesResistor 构建电阻分压链。
    • vddadiv: 调用 vddadiv_func 获取电阻网络实例，然后进行放置 (place()) 和布线 (route())。

  • 代码示例 (vddadiv_func):
    Python
    # tedhub/circuit/bgref/src/bgpref/vddadiv.py
    def vddadiv_func(vdda: Input, n: Input, uvlo_sw: Input, vdddiv: Output):
    # 定义电阻单元的参数
    parallel = ParallelResistor(RpolySingle, l=16e-6, w=1.2e-6, model="rhrp_ckt")
    series = SeriesResistor(RpolySingle, l=16e-6, w=1.2e-6, model="rhrp_ckt")
    nodes = [*Node(10)] # 创建内部节点列表
    # 实例化串并联电阻网络
    res1 = series(vdda, uvlo_sw, n=2).flipx()
    res2 = parallel(uvlo_sw, nodes[0], n=5).flipx()
    res3 = parallel(nodes[0], nodes[1], n=3)
    res4 = parallel(nodes[1], nodes[2], n=3).flipx()
    res5 = series(nodes[2], vdddiv, n=25)
    res6 = series(vdddiv, n, n=10).flipx()
    # 进行初步的行布局
    final = row(res1, res2, res3, res4, res5, res6)
    return final

  • TED 设计技巧:

    • 使用 ParallelResistor 和 SeriesResistor 简化电阻网络的创建。
    • .flipx(), .flipy(), .rotate() 用于调整单元的朝向，优化布局。
    • guardring() (在顶层模块中使用) 添加保护环，提高抗干扰能力。

2.2 PTAT 电流生成模块 (ptat.py)

  • 功能: 产生与绝对温度成正比 (PTAT) 的电流。这是带隙基准的核心部分之一。

  • 实现:

    • ptat_func: 包含一个电流镜 (cm1)，核心的 BJT 对 (b1, b2) 和相关的电阻网络 (ptres2, ptres3) 以及一些偏置和启动相关的 MOS 管 (n1, n2, n3, ptnm3)。电流镜用于复制和分配偏置电流。BJT 对的 VBE 差值通过电阻产生 PTAT 电压，进而产生 PTAT 电流。
    • ptat: 调用 ptat_func 获取各个子单元的实例，然后使用 row, col 等进行详细的版图布局，最后进行布线。

  • 代码示例 (ptat_func 结构):
    Python
    # tedhub/circuit/bgref/src/bgpref/ptat.py
    def ptat_func(...): # 输入输出端口省略
    net1, ..., net8 = Node(8) # 内部节点
    # 实例化电流镜
    cm1 = CurrentMirror(...)
    cm1.input(...)
    cm1.output(...) # 输出多路电流

    \# 实例化偏置 MOS  
net1 \>\> (n1 := NMOS(...)) % net2 \>\> net6  
net2 \>\> (n2 := NMOS(...)) % net2 \>\> net5  
\# ... 其他 MOS ...

\# 实例化电阻网络  
ptres2 \= series(net6, net4, n=6)  
ptres3 \= series(net8, net1, n=6)

\# 实例化 BJT 对  
b1 \= \[\]  
for i in range(4): (Vss(), Vss(), net4) \>\> (b := BJT(...)); b1.append(b)  
(Vss(), Vss(), net5) \>\> (b2 := BJT(...))

\# ... 创建其他 MOS 实例 ...

\# 创建 BJT 的版图结构 (自定义)  
bjts \= col(b1\[0\], b1\[1\], b2, b1\[2\], b1\[3\], spacing=-4000)  
ptbjts \= lay(...) \# 包含 bjts 和金属连接

\# 获取 MOS 和电流镜的版图  
cm1 \= guardring(cm1.layout()...)

  • TED 设计技巧:

    • CurrentMirror 模块简化了电流镜的设计和版图。
    • layout_mos 用于生成 MOS 管的版图。
    • guardring 添加保护环。
    • lay 函数和 Rect 用于创建自定义的金属连接。
    • 手动控制 BJT 的布局以优化匹配。
    • auto_rename 避免命名冲突。

2.3 带隙放大器模块 (bg_amp.py)

  • 功能: 一个运算放大器，用于建立反馈回路，强制两个关键节点的电压相等，从而稳定带隙电路的工作点。
  • 实现:
    • bg_amp_func: 实现了一个简单的差分放大器结构。包含一个差分对 (amppm1 使用 PairedMosfet)，一个有源负载电流镜 (ampcm1 使用 CurrentMirror)，以及一个输出级的 MOS 管 (m1)。
    • bg_amp: 调用 bg_amp_func 获取实例，进行行列布局 (row, col) 和布线。
  • TED 设计技巧:
    • 使用 PairedMosfet 实现匹配良好的差分输入级。
    • 使用 CurrentMirror 作为有源负载。
    • 结构清晰，易于理解和修改。

2.4 电阻网络模块 (bgref_res.py)

  • 功能: 产生最终的带隙电压 VBG 和其他分压参考。包含可 Trim 的电阻路径。
  • 实现:
    • bgref_res_func: 包含大量的串/并联电阻 (ParallelResistor, SeriesResistor) 网络，用于生成不同的参考电压 (vbg, vr1v, vr0p5v 等)。还包含由 Trim 信号 (trim_sel1 到 trim_sel6) 控制的 NMOS 开关 (m1 到 m6)，用于调整电阻值。
    • bgref_res: 获取实例，进行版图布局和布线。
  • TED 设计技巧:
    • 大量使用电阻生成辅助函数，简化复杂网络的描述。
    • 通过 MOS 开关实现电阻 Trim 功能。

2.5 Trim 逻辑模块 (ref_trim.py)

  • 功能: 存储和译码 Trim 信号。
  • 实现:
    • Latch: 基本的锁存单元。
    • Logic: 使用多个 Latch 存储外部输入的 Trim 位 (a1 到 a4)，并通过组合逻辑 (Inv, Nor2) 产生最终控制电阻网络开关的信号 (y1 到 y3, y1b 到 y3b)。
    • ref_trim_func: 包含驱动 Trim 信号的电流镜 (trimcm1, trimnmos)、一个电阻网络 (trimfq)、保险丝 (trim_pfuse) 以及 Trim 逻辑 (trimstd1, trimstd2 来自 Logic 函数)。
    • ref_trim: 调用 ref_trim_func，进行版图布局。
  • TED 设计技巧:
    • 模块化实现 Trim 功能，分为存储 (Latch) 和译码 (Logic)。
    • 使用标准单元实现组合逻辑。
    • Pfuse 器件用于永久性熔断 Trim。

2.6 UVLO 比较器 (cmp_uvlo.py) 和 温度保护 (tsc.py)

  • 功能:
    • cmp_uvlo: 检测电源电压是否低于阈值 (UVLO)，产生 uvlo 信号关闭带隙电路。
    • tsc: 检测温度是否过高 (Temperature Shutdown Circuit)，产生 temp_hib 信号。
  • 实现:
    • cmp_uvlo_func: 使用差分对 (uvlopm1) 比较输入电压和参考电压，电流镜 (uvlocm1) 作为负载，输出经过一个 MOS 管 (uvlom1) 和施密特触发器 (uvlom2) 整形。
    • tsc_func: 结构类似，使用差分对 (tscpm1) 比较 VBE (负温度系数) 和一个参考电压，电流镜 (tsccm1) 作负载，输出驱动一个 NMOS (tscm1)。
  • TED 设计技巧:
    • 利用 PairedMosfet 和 CurrentMirror 构建比较器核心。
    • 使用 SMIT (施密特触发器) 增加输出信号的抗噪声能力。

3. 顶层带隙基准集成 (bg_ref.py)

  • 功能: 将上述所有子模块 (ptat, bg_amp, bgref_res 等) 集成在一起，形成完整的带隙基准电路版图。

  • 实现:

    • bg_bjts: 一个特殊模块，用于生成匹配良好的 BJT 阵列版图，使用了手动放置 (row, col) 和金属连接 (Rect)。
    • bg_ref_func: 在 Python 层面连接各个子模块的 _func 版本，定义它们之间的信号流。注意，这里实例化的是 _func 返回的版图对象列表，而不是模块本身。
    • bg_ref: 调用 bg_ref_func 获取所有子模块的版图对象，然后执行非常复杂的手动版图布局 (col, row, lay, rotate, flipx, flipy)，将这些对象拼合成最终的版图。最后进行顶层布线 (route)。

  • 代码示例 (bg_ref 结构):
    Python

第二部分：bgp - Bandgap 生成器模式

这部分代码 (src/bgp/) 采用了一种更抽象的“生成器”模式。

  • base/ 目录: 包含各种 Bandgap 拓扑的计算类 (如 BGP_Kuijk_Basic)。这些类通常：
    • 接收设计规格 (如 Vdd, IBIAS, Vref, opamp_gain, opamp_gbw 等) 作为输入。
    • 使用查找表 (LutBjt) 或 gm/ID 模型 (GmidMos, SingleStage_FC_OTA) 进行晶体管和电阻的尺寸计算。
    • 计算出的尺寸和参数存储在类的成员变量中 (如 self.R1, self.P1_2)。
  • generator/ 目录: 包含与 base/ 对应的生成器类 (如 BGP_Kuijk_Basic_generator)。这些类通常：
    • 继承自 BGPGenerator 或 BGPGeneratorProcess。
    • 在 __init__ 中实例化一个 base 类的对象 (self.bgp = BGP_Kuijk_Basic(...)) 来获取计算好的参数。
    • 定义 core 方法：根据 self.bgp 中的参数，使用 TED 的 API (PMOS(...), NMOS(...), BJT(...), ResNet(...)) 来构建电路连接关系。
    • 通常包含一些版图模板代码 (col, row, layout_mos, guardring)，指导最终的版图生成。
    • ResNet 方法：一个辅助函数，用于根据计算出的电阻值生成相应的电阻网络版图 (使用 SeriesResistor/ParallelResistor)。
    • wrapper 方法：简单的包装，调用 build()（可能是 Generator 基类的方法）来触发 core 方法执行并生成电路。
  • tests/bgp/ 目录:
    • test_sim: 实例化生成器，调用 .test() 方法运行仿真。.test() 方法可能封装在基类 BGPGenerator 中，内部调用了 BGPTester。
    • test_layout: 调用 .lvs() 进行版图验证。
    • test_sim_post: 运行后仿真。
    • test_opt: 展示了如何结合 ted_optimization 进行电路优化。定义 problem 函数来描述优化目标和约束，然后使用优化算法 (DE, NSGA_II) 调整生成器中的 Variable 参数。
  • TED 设计技巧:
    • 计算与实现分离: base 类负责计算，generator 类负责使用 TED API 实现电路结构和版图模板。
    • 自动化流程: 这种模式易于实现自动化设计流程，根据规格自动计算参数并生成电路网表和版图。
    • 可重用性: 基类 (BGPGenerator) 和辅助函数 (ResNet) 提高了代码复用性。
    • 可优化性: 通过将关键参数定义为 Variable，可以方便地对接优化工具。

第三部分：bgp_mir - 基于电流镜和优化的版本

这部分代码 (src/bgp_mir/) 看起来是另一种带隙基准的实现，特点是：

  • 高度参数化: 大量使用了 Variable 来定义 MOS 管尺寸和电阻值，例如 self.MOS1 = (Variable(name="w1", ...), Variable(name="m1", ...)) 和 self.R1 = Variable(name="r1", ...) * 1e5。这表明该电路主要是为了进行自动化优化而设计的。
  • 结构固定: 电路的拓扑结构在 core 方法中是固定的，优化过程主要调整 Variable 定义的参数值。
  • PDK 特定实现: 文件名中包含 rpdk (如 bgp_hv_rpdk.py) 暗示可能存在针对特定 PDK 电阻模型的版本。
  • TED 设计技巧:
    • 面向优化: 将设计参数定义为 Variable，设定其边界和步长，以便优化器进行搜索。
    • 结合优化框架: 这部分代码需要配合 ted_optimization 框架使用，如 tests/bgp/test_BGP_Banba_Vbe_Linearization_generator.py 中的 test_opt 所示。

Q&A

Q1: 这个库包含了几种不同的 Bandgap 设计方法？

A1: 这个库主要展示了三种设计方法/风格：

  1. bgpref: 一个具体、完整的带隙基准实现，包含详细的手动版图布局和外围功能模块（UVLO, TSC, Trim）。适合学习完整的模块设计和版图集成流程。
  2. bgp (Generator): 一种生成器模式，将参数计算 (base) 和电路构建 (generator) 分离。提供了多种拓扑结构 (Banba, Kuijk, Voltage Mode A)，适合进行不同拓扑的快速生成和比较，也方便自动化设计。
  3. bgp_mir: 一种面向优化的设计，电路结构相对固定，但大量使用 Variable 参数化，配合优化框架使用。

Q2: base 目录中的类 (如 BGP_Kuijk_Basic) 和 generator 目录中的类 (如 BGP_Kuijk_Basic_generator) 是什么关系？

A2: base 目录中的类负责计算。它们根据输入的规格，利用器件模型（如 LutBjt, GmidMos) 计算出所需的晶体管尺寸、电阻值等参数。generator 目录中的类负责实现。它会实例化一个对应的 base 类对象来获取计算好的参数，然后调用 TED 的 API 函数（如 PMOS, NMOS, ResNet）来构建电路的连接关系和版图结构。这是一种将设计计算与电路实现分离的模式。

Q3: ResNet 函数是如何工作的？

A3: ResNet 函数 (src/bgp/generator/bgp_generator.py 或 bgp_generator_process.py) 是一个辅助函数，用于根据目标电阻值 R 和单位电阻 res_uni (特定工艺下的值) 来创建电阻网络。它首先计算出需要串联的整数单元数 k2 和需要通过串并联实现的小数部分 k1。然后，它使用 SeriesResistor 和 ParallelResistor 这两个更底层的辅助模块来搭建最终的电阻网络版图。它还处理了 R="dummy" 的情况，用于生成单位电阻的版图以填充空间或进行匹配。

Q4: 如何使用这个库进行 Bandgap 电路优化？

A4: 可以参考 tests/bgp/test_BGP_Banba_Vbe_Linearization_generator.py 中的 test_opt 函数。步骤大致如下：

  1. 选择一个 generator 类（或 bgp_mir 中的类），确保其关键参数是用 Variable 定义的。
  2. 实例化该生成器。
  3. 创建一个测试器 (BGPTester 或 CombinedTester) 来评估电路性能（如 TC, PSRR, Stability）。
  4. 定义一个 problem 函数，该函数接收测试器对象，从中提取性能指标，并返回一个 Problem 对象，其中包含优化目标 (objective) 和约束 (constraint)。
  5. 选择并配置一个优化器 (DE, NSGA_II)。
  6. 调用 tester.optimize() 传入 problem 和优化器配置。
  7. 调用 opt.solve() 运行优化。
  8. 处理优化结果，例如应用最优参数并重新仿真验证。

Q5: bgpref 和 bgp 的代码风格有何不同？哪个更推荐？

A5:

  • bgpref 风格更接近传统的手动设计流程，对版图有精细的控制，适合需要高性能和精确匹配的最终产品设计。但开发和修改可能较慢。
  • bgp (Generator) 风格更自动化，适合快速生成不同拓扑或根据规格自动 sizing，便于早期探索和设计空间评估。版图控制相对模板化。
  • bgp_mir 风格专注于优化，电路结构固定，通过调整参数达到目标。

选择哪种风格取决于您的需求。如果是学习 TED 的完整设计流程和手动版图技术，bgpref 是很好的例子。如果需要快速生成多种设计或进行自动化 sizing 和优化，bgp 或 bgp_mir 的思路更合适。实践中也可能结合使用，例如用生成器快速得到初步设计，再进行手动版图优化。
