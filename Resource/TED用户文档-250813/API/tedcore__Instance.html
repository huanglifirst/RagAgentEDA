def add_hbus(
    self: "Instance",
    *args: Union["CircuitNode", str],
    layer: Optional["Layer"] = None,
    vias: int = 1,
    width: int = None,
    location: int = BOTTOM,
    spacing: int = 300,
    bus_spacing: Optional[int] = None,
    priority: int = None,
    extend: Tuple[int, int] = (CENTER, 0),
    overlap: bool = False,
    to_origin: bool = True,
    auto_cut: bool = False,
) -> "InstList":

添加水平的 bus

参数

序号参数说明类型默认值
1argsbus 连接的节点，传入几个节点就生成几根 bus。bus 顺序与节点顺序一致Union['CircuitNode', str]无
2layer层级Optional['Layer']None
3viasbus 宽度，用打孔数量描述，不能与 width 参数同时指定int1
4widthbus 宽度，用实际大小表示，不能与 vias 参数同时指定intNone
5locationbus 相对实例的位置，只能为 LEFT, CENTER, RIGHTint0
6spacingbus 与实例之间的间距int300
7bus_spacingbus 之间的间距，为 None 表示取层级的最小间距Optional[int]None
8prioritybus 的优先级，高优先级布线器会优先连接intNone
9extend扩展的大小。如 (TOP, 100) 表示向上扩展 100；(MIDDLE, 100) 表示向上向下各扩展 100Tuple[int, int](4, 0)
10overlap是否重叠摆放，如果为 True 会使得 spacing 失效boolFalse
11to_origin是否移动到原始位置boolTrue
12auto_cut是否自动裁剪。若设置extend则会导致cut失效boolFalse

返回值

序号说明类型
1实例与 bus 的组成的整体InstList
