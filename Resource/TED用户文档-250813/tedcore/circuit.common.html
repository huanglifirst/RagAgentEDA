watch

功能介绍:

watch 是一个用于在电路仿真过程中监控特定信号的辅助函数。它主要用于调试和分析仿真结果，在运行仿真（如 tran）后，指定需要观测的信号名称，以便后续通过 raw.get_signal() 或 debug 等方式获取其波形数据。

该函数是全局函数，内部调用底层电路对象（get_circuit()）的 watch 方法，并将传入的信号名追加到 watch_signals 列表中，用于记录需要监控的信号。此功能适用于所有类型的电路仿真场景，尤其是复杂的模拟电路设计验证。

参数定义:

参数名类型含义默认值
*argsstr 的可变长度参数列表要观察的信号名称。可以是节点名、模块实例输出端口名或探针（probe）名称等。无
**kwargsdict额外的信号监控参数，如仿真配置选项等。具体支持的参数由底层实现决定。无

返回值

无返回值。

skip_unless

功能介绍:

skip_unless 是一个用于条件性跳过测试用例的装饰器函数，主要用于在自动化测试环境中根据配置条件动态决定是否执行某个测试。该函数通过检查多个运行时环境条件（如测试级别、工艺库、仿真器类型、环境变量等）来判断当前测试是否应被跳过。

该函数设计为可被 pytest 框架识别，当所有检查条件均满足时，测试正常运行；若任一条件不满足，则自动标记测试为“跳过”，并提供对应的跳过原因说明。此机制有助于实现测试的分级执行控制，尤其适用于耗时较长或依赖特定环境资源的测试。

参数定义:

参数名类型含义默认值
TED_TEST_LVLOptional[int]指定测试用例预期运行时间（单位：分钟），用于控制仅在环境变量 TED_TEST_LVL 大于等于该值时才执行。无
TED_PDKUnion[List[str], str]指定测试用例所依赖的工艺库（PDK）名称，支持单个或多个工艺库。无
TED_SKIP_PDKUnion[List[str], str]指定应跳过的工艺库集合，作为 TED_PDK 的补集，用于忽略某些特定工艺库下的测试。无
simulatorOptional[Union[str, List[str]]]指定测试所使用的仿真器名称，支持单个或多个仿真器。若当前运行环境中未匹配指定仿真器，则跳过测试。无

needs_layout

功能介绍:

needs_layout 是一个用于判断是否需要生成版图（layout）的辅助函数。其核心逻辑是检查传入的参数中是否包含 Variable 类型的对象。如果所有参数均不包含 Variable 实例，则返回 True，表示需要生成版图；否则返回 False，表示无需生成版图。

该函数主要用于在电路设计流程中判断当前上下文是否涉及可优化变量（即 Variable），避免在存在优化变量的情况下重复执行版图生成功能。函数支持通过位置参数和关键字参数传入任意多个参数，并可在无显式参数时自动检测调用它的函数的本地作用域中的局部变量。

注：该函数是 TED 框架中用于控制版图生成逻辑的重要工具，特别适用于模块化电路设计流程中对生成行为的智能决策。

参数定义:

参数名类型含义默认值
*args可变位置参数（任意类型）用于传入多个位置参数，用于判断是否包含 Variable。无
**kwargs可变关键字参数（任意类型）用于传入多个关键字参数，用于判断是否包含 Variable。无

返回值

  • 类型：bool
  • 含义：若所有输入参数均不包含 Variable 实例，则返回 True，表示需要生成版图；否则返回 False，表示无需生成版图。

  • 隐式调用（无参数）：自动分析调用该函数的上层函数的局部变量（通过 inspect），判断其中是否含有 Variable。

  3. 注入配置至 DUT 模型：将获取到的 stage 值传入 dut.test() 方法，实现参数化仿真。同时使用其他固定参数（如 OTAtype='siso' 或 'tia'、simulator='spectremdl'）完成完整仿真流程。

综上，getenv 用于在运行时动态读取电路全局配置项，支持模块化的电路生成和可复用的仿真脚本设计。

auto_rename

功能介绍:

auto_rename 是一个用于自动重命名当前模块的辅助函数，主要在电路设计上下文（基于 TED 框架）中使用。该函数通过分析调用栈信息，获取当前执行环境中的函数调用路径，并结合当前模块信息，自动为当前正在定义或实例化的模块生成合适的名称。

该功能适用于在嵌套或动态创建模块的场景中，确保每个模块具有唯一且语义清晰的命名，避免命名冲突或难以追踪的问题。其行为依赖于 current_module() 和栈帧（frame）分析机制。

参数定义:

参数名类型含义默认值
backint表示从调用 auto_rename 的位置向上回溯的栈帧层数。用于定位当前上下文所在的函数层级。0

返回值

无返回值（None）。

        assert current_module() is None
        auto_rename(back=0)
        assert current_module() is not None

        assert current_module().name == 'inner_func'

    def outer_func():
        inner_func()

    outer_func()
