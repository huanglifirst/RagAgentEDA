ParameterMixin

功能介绍:

ParameterMixin 是一个混合类（mixin），为 TED 框架中的实例对象（Instance）提供对模块参数的管理能力。该类允许用户获取、设置和查询与实例相关的参数，支持参数值的类型转换，并可将所有参数以字符串形式组合返回。此功能主要用于在电路设计中动态访问和操作模块配置参数。

参数定义:

参数名类型含义默认值
paramstrstr要设置的参数字符串，通常是键值对组成的格式，用于初始化或更新实例参数。无
namestr参数名，用于查询某个特定参数的值。无
original_typebool是否返回原始类型（非字符串形式），若为 True，则将存储的字符串值按原始类型还原为整数、浮点数、布尔或字符串。False
skip_hashcodebool是否跳过哈希码（HASHCODE_STR）参数。当 True 时，返回结果中不包含哈希码项。False

返回值

  • set_paramstr: 返回 self，支持链式调用。
  • get_parameter: 返回类型为 Optional[Union[int, float, bool, str]]，表示参数的值，若不存在或为空则返回 None。
  • get_parameter_str: 返回类型为 str，返回指定参数对应的字符串值；若不存在或为空，则返回 None。
  • has_parameter: 返回类型为 bool，表示当前实例是否包含指定名称的参数。
  • get_all_parameters: 返回类型为 Dict[str, str]，包含所有参数名和其对应字符串值组成的字典。
  • get_all_func_parameters_str: 返回类型为 str，返回所有非哈希码参数以函数调用格式拼接成的字符串（如 key1=value1, key2='value2'）。

代码示例:

def test_set_paramstr():
    inst = Instance()
    inst.set_paramstr("width=0.5, height=1.0")
    assert inst.get_parameter("width") == 0.5
    assert inst.get_parameter("height") == 1.0

def test_get_parameter():
    inst = Instance()
    inst.set_paramstr("is_active=true, scale=2.5")
    assert inst.get_parameter("is_active") is True
    assert inst.get_parameter("scale") == 2.5
    assert inst.get_parameter("nonexistent") is None

def test_get_parameter_str():
    inst = Instance()
    inst.set_paramstr("layer=Layer(1,2)")
    assert inst.get_parameter_str("layer") == "Layer(1,2)"

def test_has_parameter():
    inst = Instance()
    inst.set_paramstr("vdd=3.3")
    assert inst.has_parameter("vdd") is True
    assert inst.has_parameter("gnd") is False

def test_get_all_parameters():
    inst = Instance()
    inst.set_paramstr("width=10, height=20, scale=1.5")
    params = inst.get_all_parameters(original_type=True, skip_hashcode=True)
    assert params == {"width": "10", "height": "20", "scale": "1.5"}

def test_get_all_func_parameters_str():
    inst = Instance()
    inst.set_paramstr("width=10, height='abc', layer=Layer(1,2)")
    result = inst.get_all_func_parameters_str()
    assert result == "width=10, height='abc', layer=Layer.get_layer_from_number(1,2)"

set_paramstr

功能介绍:

set_paramstr 是一个用于设置实例参数字符串的方法，属于 Instance 类的成员函数。该方法接收一个字符串参数，用于配置底层电路模型的行为或特性（例如电源信号的波形类型、时序参数等），并将其传递给内部的电阻/电源实例（rsinst）进行实际应用。

该方法主要用于在电路仿真中定义如脉冲波形（pulse）、直流电压（dc）、上升沿/下降沿时间（rise/fall）、周期和占空比等动态信号行为。通过调用此函数，用户可以灵活地为电路中的电源节点或激励信号设置复杂的时序参数。

参数定义:

参数名类型含义默认值
paramstrstr用于配置参数的字符串，格式遵循仿真工具（如SPICE）的语法规范，例如设置脉冲信号时包含 type=pulse dc=0 val0=0 delay=0 val1=1.2 width=20u period=40u rise=1n fall=1n 等参数。无

返回值

返回调用该方法的当前 Instance 实例本身（即 self），支持链式调用。

has_parameter

功能介绍:

该函数用于判断一个模块实例（Instance）中是否包含指定名称的参数。
在 TED 框架中，模块（Module）可以定义具有默认值的参数，这些参数在实例化时可被设置或继承。has_parameter 方法允许开发者在运行时检查某个给定名称的参数是否存在。

这是一个实例方法，需通过 Instance 对象调用，其底层实际委托给 module().has_parameter(name) 实现，即查询该实例所对应模块是否定义了指定名称的参数。

参数定义:

参数名类型含义默认值
namestr要检查的参数名称。无

返回值

  • bool：若实例对应的模块中包含指定名称的参数，则返回 True；否则返回 False。

代码示例:

def test_has_parameter():
    @module
    def buffer(
        A: Input,
        Y: Output,
        width: int = 1,
        enable: bool = True,
    ):
        return buffer_inst()

    inst = buffer(A=A, Y=Y)
    
    assert inst.has_parameter("width") is True
    assert inst.has_parameter("enable") is True
    assert inst.has_parameter("invalid_param") is False

说明：

  • 示例中定义了一个名为 buffer 的模块，包含两个参数：width（默认值为 1）和 enable（默认值为 True），且均有默认值。
  • 通过 buffer(A=A, Y=Y) 创建一个实例 inst。
  • 调用 inst.has_parameter("width") 和 inst.has_parameter("enable")，返回 True，说明这两个参数是存在的。
  • 调用 inst.has_parameter("invalid_param") 返回 False，表明该名称并非模块定义中的参数。

此方法适用于在动态配置或条件逻辑中判断某个参数是否可用。

get_parameter_str

功能介绍:

该函数用于从实例（Instance）中获取指定名称的参数值，并以字符串形式返回。它首先调用底层接口 get_rsinst().get_parameter_str(name) 获取原始参数值，若返回值为 NONE_STR（表示未定义或空），则将其转换为 None；随后检查该参数名是否存在于当前电路（Circuit）的变量字典中，若存在，则取出对应的变量值并转换为字符串返回。此函数主要用于在电路设计过程中，安全地读取配置参数的字符串表达。

该接口是 Instance 类的一个成员方法，用于获取特定参数的字符串表示形式，适用于需要动态解析、调试或构建配置信息的场景。

参数定义:

参数名类型含义默认值
namestr要查询的参数名称。无

返回值

  • 类型：str 或 None
  • 含义：返回指定参数名称对应的字符串值；如果参数未定义或在变量表中不存在，则返回 None。

代码示例:

def test_get_parameter_str():
    # 创建一个实例并配置参数
    inst = Instance(
        parameter_a="hello",
        parameter_b=42,
        parameter_c=None
    )

    # 调用 get_parameter_str 获取字符串参数
    result1 = inst.get_parameter_str("parameter_a")
    assert result1 == "hello"

    # 测试变量表中存在且有值的情况
    from tedcore.kernel import get_circuit
    circuit = get_circuit()
    circuit.variables["parameter_b"] = 99
    result2 = inst.get_parameter_str("parameter_b")
    assert result2 == "99"

    # 测试未定义的参数
    result3 = inst.get_parameter_str("nonexistent_param")
    assert result3 is None

    # 测试参数值为 NONE_STR 的情况
    inst.set_parameter("parameter_c", NONE_STR)
    result4 = inst.get_parameter_str("parameter_c")
    assert result4 is None

get_parameter

功能介绍:

get_parameter 是一个实例方法，用于从电路模块实例中根据参数名称获取对应的参数值。该方法支持返回整数、浮点数、布尔值和字符串等多种数据类型，并能自动进行类型转换。当参数的原始类型为字符串且未映射到特定类型时，直接返回原字符串；若参数值为特殊标记 NONE_STR（表示空值），则返回 None。

此方法适用于所有继承自 Instance 类的实例对象，如 DelayedInstance、MOS 管实例等，在电路仿真或配置查询中用于读取模块的参数设置。

参数定义:

参数名类型含义默认值
namestr需要获取值的参数名称。无

返回值

  • 返回类型为 Optional[Union[int, float, bool, str]]
  • 表示可能返回整数、浮点数、布尔值或字符串中的任意一种类型，也可能返回 None（当参数值为空时）

代码示例:

# /home/zuochang/ted/tedcore/kernel/src/instance/tests/test_delayed.py:47

        assert mos.get_parameter('l') == 1e-06

        assert mos.get_parameter('m') == 2

  • 调用 initialize() 后得到一个 mos 实例对象。
  • 使用 get_parameter('l') 获取参数 l 的值，返回 1e-06（即 1 微秒），对应延迟时间。
  • 使用 get_parameter('w') 获取宽度参数的值，同样为 1e-06。
  • 使用 get_parameter('m') 获取尺寸因子，由于在 set_params 中显式设为 2，因此返回 2。

该用例展示了 get_parameter 方法在获取不同类型参数值时的行为：

  1. 数值型参数：如 'l' 和 'w'，返回浮点数（float）；
  2. 整型参数：如 'm'，返回整数（int）；
  3. 系统自动识别参数类型并转换，无需手动处理字符串到数值的转换。

此方法可安全用于读取任意已配置模块的参数值，是电路设计中常见查询与调试手段。

get_all_parameters

功能介绍:

该函数用于获取实例（Instance）所对应模块的所有参数信息，返回一个包含所有参数名及其值的字典。

  • 如果 original_type 参数设为 True，则会将原本以字符串形式存储的参数类型还原为其原始类型（如整数、浮点数等）。
  • 如果 skip_hashcode 参数设为 True，则在返回结果中排除哈希值相关的参数（如自动生成的唯一标识）。

该方法是实例类的一个成员函数，其功能基于对应的模块（Module）提供的 get_all_parameters 方法实现。

参数定义:

参数名类型含义默认值
original_typebool是否还原参数的原始类型；当为 True 时，将字符串格式的参数值转换为原始数据类型。False
skip_hashcodebool是否跳过（排除）哈希值相关的参数；当为 True 时，结果中不包含与哈希码相关的参数。False

返回值

  • 类型：Dict[str, str]
  • 含义：一个字典，键为参数名，值为参数的字符串表示形式。若启用了 original_type=True，则值会以原始数据类型呈现（但在函数内部仍以字符串形式传递，实际类型由底层逻辑转换）。

代码示例:

    assert len(result) >= 0

    assert isinstance(result, dict)

    inst = Instance()
    result = get_all_parameters(inst, skip_hashcode=True)
    assert isinstance(result, dict)

# 测试用例4：同时启用 original_type 和 skip_hashcode
def test_get_all_parameters_both():
    inst = Instance()
    result = get_all_parameters(inst, original_type=True, skip_hashcode=True)
    assert isinstance(result, dict)

说明：

  • test_get_all_parameters_default 用于验证在不进行任何额外处理的情况下，能够正确返回所有参数的字符串形式。
  • test_get_all_parameters_original_type 展示了如何将存储的字符串值还原为原始类型（例如数值、布尔值等），适用于需对参数做类型判断或计算的场景。
  • test_get_all_parameters_skip_hashcode 用于排除由系统自动生成的哈希参数，减少输出冗余，常用于配置导出或比对场景。
  • test_get_all_parameters_both 综合展示了两个选项同时启用的情况，适用于需要清洗数据并还原类型的情形。

get_all_func_parameters_str

功能介绍:

该函数用于获取实例（Instance）在调用时所使用的所有参数，并以字符串形式返回。返回的字符串格式为 key1=value1, key2=value2, ...，其中每个键值对表示一个参数及其对应的值。该函数适用于描述实例化过程中的配置状态，便于调试或日志输出。

  • 如果参数值是数字、None（对应常量 NONE_STR）、True 或 False（对应常量 TRUE_STR 和 FALSE_STR），则直接以原值形式表示。
  • 如果参数值是以 'Layer' 开头的字符串（如 'Layer(1,2)'），则将其转换为 Layer.get_layer_from_number(layer, datatype) 的调用形式。
  • 若参数键为 HASHCODE_STR，则忽略该键不参与返回字符串。
  • 所有参数按其在字典中出现顺序拼接。

此函数主要用作调试或信息展示，帮助查看某个实例在创建时传递了哪些具体参数。

参数定义:

参数名类型含义默认值
selfInstance当前实例对象，用于获取其所有参数。无

返回值

  • 类型: str
  • 含义: 以逗号分隔的键值对字符串，表示该实例调用时的所有非哈希码参数及其对应值。格式为 key1=value1, key2=value2, ...
