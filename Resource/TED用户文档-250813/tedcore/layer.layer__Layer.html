Layer

功能介绍:

Layer 类用于表示半导体工艺中的一个层级（如金属层、接触层等），是电路设计中描述物理布局结构的核心数据结构之一。该类封装了与工艺层相关的关键属性，包括层号、数据类型（datatype）、最小宽度、间距规则以及与其他层之间的上下文关系。通过此对象可访问和管理层的电气与几何特性，并支持层之间的顺序比较、路径获取等操作。

Layer 对象具备丰富的比较运算符重载功能，能够实现基于层号和数据类型的大小判断。此外，提供了用于计算最小线宽（考虑电流密度）、最大宽度以及获取上/下层金属连接宽度的方法，适用于在多层金属设计中自动推导布线参数。

特别地，__rshift__ 方法实现了从当前层到目标层的路径遍历功能，返回中间所有层的有序列表，可用来获取两个金属层之间的完整堆叠路径（含闭区间）。此功能常用于自动化DRC检查、布线策略分析或层次化设计流程。

参数定义:

参数名类型含义默认值
lyLayerInfo 或其他底层层描述对象代表实际工艺层信息的对象，包含层号、datatype、几何规则等。无
is_original_namebool标记该层名称是否为原始命名（如未经过规范化处理），影响字符串输出格式。False

返回值

返回值类型含义
str以字符串形式返回当前层的标识（通过调用底层对象的 to_str() 方法）。
bool所有比较操作符（==, !=, <, >, <=, >=）均返回布尔类型表示逻辑关系。
intget_min_width, get_min_diagonal_width, get_max_width, get_width 等方法返回整型数值，如最小宽度、最大宽度等。
LayerList__rshift__ 操作符返回一个 LayerList 类型对象，表示从起始层到目标层之间的所有中间层（含两端）。
set[int]get_width 中使用 set 存储从通孔连接中提取出的所有可能宽度值（水平与垂直方向）

代码示例:

def test_layer_comparison():
    AA = Layer("AA")
    M1 = Layer("M1")
    M2 = Layer("M2")

    assert AA < M1
    assert M1 < M2
    assert AA <= M2
    assert M2 > M1
    assert not (AA > M1)
def test_layer_path():
    AA = Layer("AA")
    M1 = Layer("M1")
    M2 = Layer("M2")

    result = AA >> M2
    assert isinstance(result, LayerList)
    assert list(result) == ["AA", "M1", "M2"]

    result = M2 >> AA
    assert isinstance(result, LayerList)
    assert list(result) == ["M2", "M1", "AA"]
def test_get_width():
    M1 = Layer("M1")
    # 假设 M1 是金属层，且有对应的上/下层和通孔规则
    widths = M1.get_width(via=1, level="all")
    assert isinstance(widths, set)
    for w in widths:
        assert isinstance(w, int) and w > 0
def test_min_width_with_current():
    M1 = Layer("M1")
    # 假设其能承受一定电流密度
    min_width = M1.get_min_width_by_current(current=0.01)
    assert isinstance(min_width, int) and min_width > 0
def test_get_layer_properties():
    AA = Layer("AA")
    min_w = AA.get_min_width()
    diag_w = AA.get_min_diagonal_width()
    max_w = AA.get_max_width()

    assert isinstance(min_w, int) and min_w >= 0
    assert isinstance(diag_w, int)
    assert isinstance(max_w, int)

说明:

  • Layer 支持使用 <, >, == 等运算符进行层间比较，按照“层号 > 数据类型”的规则排序。

功能介绍:

函数主要实现逻辑为：

  • 依据传入的通孔数量 via，通过组合分解得到行数与列数的组合；
  • 针对每组行列组合（row, col），调用底层方法 get_bboxs_consider_en 获取符合封装要求的矩形区域；
  • 将所有结果构造成 Box 对象并返回。

注意：该函数不考虑上下金属层的最小尺寸限制，仅基于通孔数量和封装规则生成矩形候选。如果当前对象不是通孔层，则会抛出断言错误。

get_all_layers

功能介绍:

get_all_layers 是一个静态方法，用于获取系统中所有的版图层（Layer）信息。该方法支持按原始 PDK 名称获取层，也可以通过自定义过滤函数对层进行筛选。

  • 当 is_original=False（默认）时，返回的是使用 TED 系统内部命名的层。
  • 当 is_original=True 时，返回的是来自 PDK（工艺设计套件）的原始层名称。
  • 可通过传递一个可调用函数 func 来实现对所有层的自定义过滤逻辑。若未提供，则默认保留所有层。

该方法最终返回一个 LayerList 类型的对象，其中包含了满足条件的所有 Layer 对象，可用于后续对版图层的遍历、筛选或操作。

参数定义:

参数名类型含义默认值
is_originalbool是否使用 PDK 中的原始层名称。若为 True，则从 PDK 原始数据中获取层；否则使用 TED 内部命名。False
funcOptional[Callable[['Layer'], bool]]可选的过滤函数，输入为一个 Layer 对象，返回布尔值。若返回 True，则将该层加入结果列表。若未提供，则默认保留所有层。None

返回值

  • 类型: LayerList
  • 含义: 包含所有符合条件的 Layer 实例的一个列表对象，可用于后续遍历或进一步处理。

check_spacingtable

功能介绍:

该函数用于根据间距表（spacing table）规则计算两个几何图形之间的最小所需间距。此方法适用于在集成电路设计中进行物理验证，确保不同金属层或导线之间满足工艺要求的最小间距标准。
该函数属于 Layer 类的成员方法，其核心功能是调用底层数据库接口（self.ly.check_spacingtable）完成具体的间距查表计算，并对输入参数进行整数化处理以保证精度。

参数定义:

参数名类型含义默认值
paralint并行运行长度，即两个几何图形在平行方向上的延伸长度。无
width1int第一个几何图形的宽度。无
width2int第二个几何图形的宽度。无
length1Optional[int]第一个几何图形的长度，若未提供则默认等于其宽度 width1。None
length2Optional[int]第二个几何图形的长度，若未提供则默认等于其宽度 width2。None

返回值

  • 返回类型：int
  • 含义：根据间距表规则计算出的最小所需间距（单位为数据库单位，即DBU）。
