MOSFET器件库 (NMOS/PMOS/...)

1. 概述

ted_device.components.mos 模块是 TED 中用于创建、布局和描述 MOSFET 晶体管的核心。它在 ted_builtins 提供的基础仿真/LVS 模型之上，增加了参数化的版图生成能力和对常见模拟设计技术的支持。

主要功能包括：

  • 标准 MOSFET: 提供 NMOS 和 PMOS 类，用于实例化基本的 N 型和 P 型晶体管。
  • 版图生成: 能够根据参数（L, W, M, model 等）自动生成符合 PDK 规则的版图，支持源漏共享、多 finger 等。可以选择使用 TED 内部逻辑生成 (from_virtuoso=False) 或尝试导入 Virtuoso Pcell 的 GDS (from_virtuoso=True)。
  • 高级布局结构: 提供用于差分对等匹配应用的特殊布局生成器，如 PairedMosfet（支持 ABBA、Cross-Coupled 布局）、ABBA 模块等。
  • Dummy 器件: 提供了 Dummy 模块来生成哑元管，用于改善匹配性或满足密度规则。
  • 阵列/网格: MosGrid 用于生成大规模的 MOS 阵列。
  • 特殊器件: 支持 5 端 (PMOS5T) 和 6 端 (NMOS6T) 的特殊 MOSFET。
  • 辅助功能: 提供了丰富的属性访问 (.l(), .w(), .model() 等) 和方法 (.get_ds(), .singularize_mos() 等) 以及布局助手 (layout_mos, row_mos, guardring)。

2. 核心 MOSFET 类 (mos.py)

NMOS 和 PMOS 是用户最常直接使用的类。

  • NMOS(g: Ctrl, d: Input, s: Output, l=None, w=None, nf=None, m=None, model=None, from_virtuoso=None, obs=None, builtin=True, **params):
    • 实例化一个 N 型 MOSFET。
    • 连接: 使用 d >> nmos % g >> s 的方式连接漏极 (d)、栅极 (g) 和源极 (s)。体端 (b) 默认连接到 Vss()。
  • PMOS(g: Ctrl, d: Input, s: Output, l=None, w=None, nf=None, m=None, model=None, from_virtuoso=None, obs=None, builtin=True, **params):
    • 实例化一个 P 型 MOSFET。
    • 连接: 使用 d >> pmos % g >> s 的方式连接。体端 (b) 默认连接到 Vdd()。
  • 主要参数:
    • l, w: 沟道长度和宽度 (单位: 米)。若不指定，使用 get_mosfet_default_l/w(model) 获取 PDK 默认值（通常为 Lmin 和 3*Lmin 或 Wmin 中的较大者）。
    • m, nf: 乘数 (Multiplier) 或 Finger 数量。两者等效，不能同时指定。m=1 (或 nf=1) 表示单 finger 器件。
    • model: PDK 中的 MOS 模型名称 (如 "nch", "pch_lvt")。若不指定，使用 getenv("nch") 或 getenv("pch") 获取默认模型。
    • from_virtuoso: (bool) 是否尝试从 Virtuoso 获取版图 (True) 或使用 TED 内建版图生成 (False)。默认值由 getenv("mos_from_virtuoso") 决定。
    • obs: (Layer) 版图生成时需要避开的障碍层。
    • builtin: (bool) 是否自动包含对应的 LVS 和 Sim 模型 (MosfetLvs, Mosfet 或 MosfetFromVirtuoso*)。通常保持 True。
  • 内部实现:
    • 调用 Mos 基类。
    • 根据 from_virtuoso 的值，选择调用 MosLayoutArray (TED 生成) 或 MosFromVirtuosoLayout (导入 GDS) 来创建版图视图。
    • 调用 ted_builtins 中的相应模块来创建 LVS 和 Sim 视图。

示例 (基本 NMOS/PMOS 实例化 - 改编自 test_mos.py::test17/18/19)

from ted_device import NMOS, PMOS, Mos
from tedcore import Node, Vdd, Vss, testbench, place

@testbench
def BasicMosFET():
    d_n, g_n, s_n = Node(3, name="n") # d, g, s for NMOS
    d_p, g_p, s_p = Node(3, name="p") # d, g, s for PMOS

    # 实例化 NMOS (l=1u, w=5u, m=2 fingers)
    nmos1 = NMOS(l=1e-6, w=5e-6, m=2, model="nch")
    d_n >> nmos1 % g_n >> s_n

    # 实例化 PMOS (使用默认 L/W, m=1)
    pmos1 = PMOS() # model 会使用 getenv("pch")
    d_p >> pmos1 % g_p >> s_p

    # 验证参数和连接
    assert nmos1.l() == 1e-6
    assert nmos1.w() == 5e-6
    assert nmos1.m() == 2
    assert nmos1.model() == "nch"
    assert nmos1.drain() == d_n
    assert nmos1.gate() == g_n
    assert nmos1.source() == s_n
    assert nmos1.body() == Vss() # NMOS body connects to Vss

    assert pmos1.model() == getenv("pch")
    assert pmos1.body() == Vdd() # PMOS body connects to Vdd

    place(col(nmos1, pmos1)) # 布局

3. 版图生成与布局

ted_device.mos 模块提供了强大的版图生成能力。

3.1 版图生成选项

  • TED 内部生成 (from_virtuoso=False, 默认):
    • MosLayoutArray: 处理 m > 1 的情况，通过排列 MosLayout 实例并自动处理源漏共享。
    • MosLayout: 生成单个 MOS (m=1) 的版图，使用 MosFields 获取 PDK 几何参数。
    • 优点：纯 Python 实现，不依赖外部工具，速度快。
    • 缺点：需要 MosFields 提供准确的几何参数 (通过 PDK YAML 文件配置)。
  • 从 Virtuoso GDS 导入 (from_virtuoso=True):
    • MosFromVirtuosoLayout:
      • 调用 ted_interface.export_gds_from_virtuoso 导出指定 model 和参数的 Pcell GDS 文件。
      • 读取 GDS 文件内容 (read_flattened_gds_to_shape)。
      • 识别 D/G/S 端口层 (recognize_mos_base_nboxes)。
      • 创建对应的 TED 实例。
    • 优点：直接使用 PDK 厂商提供的 Pcell 版图，保真度最高。
    • 缺点：需要配置好 Virtuoso 环境和接口，速度较慢。

3.2 多 Finger 与源漏共享 (mos_layout_array.py, mos_algorithm.py)

  • 当 m > 1 时，MosLayoutArray 会创建 m 个 MosLayout 实例。
  • row_mos(mosfets, align) 辅助函数：
    • 自动计算相邻 MOS 间的重叠距离 (get_overlap_size_by_two_mos) 以实现源漏共享。
    • 使用 row_with_different_spacing 进行紧凑排列。
    • 处理奇偶 m 值和 flipy() 变换后的端口连接。

示例 (使用 row_mos 布局串联 MOS - 改编自 test_mos_algorithm.py::test06)

@testbench
def RowMosExample():
    n1, n2, n3, g = Node(4)
    # 创建两个 NMOS
    mos1 = NMOS(l=1e-6, w=2e-6, m=1)
    mos2 = NMOS(l=1e-6, w=2e-6, m=1)

    # 连接： n1 -> mos1 -> n2 -> mos2 -> n3
    n1 >> mos1 % g >> n2
    n2 >> mos2 % g >> n3

    # 使用 row_mos 布局，mos1 的源极和 mos2 的漏极会自动共享
    layout = row_mos([mos1, mos2])
    place(layout)

    # 验证共享: mos1 的右侧(源极) M1 矩形与 mos2 的左侧(漏极) M1 矩形是同一个
    # 注意：由于内部实现细节，直接比较 Instance 可能不等，比较 Box 是否重合更可靠
    assert mos1.get_right_rect().box.is_coincident(mos2.get_left_rect().box)

3.3 保护环 (guardring) 和外围环 (mos_algorithm.py)

  • guardring(inst, connectTo, type, ...) : （来自 layout 模块）为器件或 InstList 添加保护环。
  • mos_outer_ring(inst, model, keepout): 根据 PDK 中 model 的 ring_layer.outer 配置，在 inst 外围添加指定的层次（如 NWELL, DNW）。
  • mos_inner_ring(inst, model): 根据 PDK 中 model 的 ring_layer.inner 配置，在 inst 内部添加指定的层次。

3.4 布局助手 (layout_mos)

  • layout_mos(mosfets, supply, tap, connect_single_ds, part_max_m, s_shaped, ...): （来自 algo 模块）一个强大的高层布局函数，用于自动排列和连接 MOS 管列表。
    • 支持源漏共享、Dummy 管插入、Tap 单元添加、S 型走线、保护环集成等。
    • 是构建更复杂结构（如 PairedMosfet, CurrentMirror）的基础。

示例 (使用 layout_mos - 改编自 test_mos_algorithm.py::test_layout_mos)

@testbench
def LayoutMosExample():
    g, d, s = Node(3)
    # 创建一个 m=3 的 PMOS
    pmos1 = PMOS(l=1e-6, w=5e-6, m=3, model="pch")
    d >> pmos1 % g >> s

    # 使用 layout_mos 进行布局，自动处理内部连接
    # supply=0: 不自动连接 D/S 到 VDD/VSS (已手动连接)
    # tap=0: 不添加 Tap 单元
    # connect_single_ds=True: 对于 m=1 的管子也强制连接 D/S 到外部（通常用于阵列）
    pmos_layout = layout_mos(mosfets=[pmos1], supply=0, tap=0, connect_single_ds=True)

    place(pmos_layout)
    # route(pmos_layout) # layout_mos 内部通常已包含走线

4. 特殊 MOSFET 结构

4.1 差分对管 (PairedMosfet) (components/mos/paired_mos)

  • 功能: 生成用于差分对的匹配性布局。
  • 模式 (mode):
    • "abba": ABBA 排列。
    • "cross": 交叉耦合（使用 M1/M2）。
    • "cross_high_layer": 交叉耦合（使用更高层金属，如 M2/M3）。
  • 参数:
    • d1, d2, g1, g2, s: 差分信号端口。
    • dummy_m, dummy_l: Dummy 管参数。
    • y_spacing: (用于 cross* 模式) 上下两半之间的垂直间距。
    • d_via: 漏极连接使用的 Via 数量。
  • 特殊版本: PairedNMosfet5T, PairedNMosfet6T 用于 5/6 端子器件。

示例 (见 ted_device 文档第 9 节的详细样例)

4.2 ABBA 布局 (components/mos/abba.py, abba1.py)

  • ABBA(d1, d2, g1, g2, s1, s2, ...): 标准 ABBA 布局，两个源极分开。
  • ABBA_CO(d1, d2, g1, g2, s, ...): Common Output，两个源极连接到同一个 s 节点。
  • ABBA_CI(d, g1, g2, s1, s2, ...): Common Input，两个漏极连接到同一个 d 节点。
  • NMOS_ABBA_CO_VSS, PMOS_ABBA_CO_VDD, PMOS_ABBA_CI_VDD: 特定类型且源/漏直接连接到电源轨的 ABBA 结构。

4.3 Dummy 器件 (components/mos/dummy.py)

  • Dummy(node, l, w, m, model, ...): 生成指定数量 (m) 的 Dummy MOS 管。
    • 栅极和源/漏极通常连接到电源/地 (supply = get_vsupply(model))。
    • node: 暴露的一个连接点（通常是外侧的源/漏）。
    • h_gate, v_gate: 控制连接栅极的 Via 大小。
    • add_m2: 是否添加 M2 连接层。
  • create_outer/inner_dummy_list(...): 创建 Dummy 列表的辅助函数。
  • setup_dummy_node(mosfets, dummy_tag): 处理包含带有 dummy_tag 的 MOS 管列表的连接。

4.4 MOS 网格 (components/mos/mos_grid.py)

  • MosGrid(g, d, s, l, w, m, rows, model, add_guardring): 创建 rows 行 x m 列的 MOS 管阵列。
  • MosGridElement(..., rows<=2, ...): MosGrid 的基本构建块（最多两行）。

示例 (改编自 test_mos_grid.py::test_mosgrid1)

@testbench
def MosGridExample():
    g, d, s = Node(3)
    lmin = get_pdk().lmin()
    # 创建一个 4 行 x 5 列的 PMOS 网格，并添加保护环
    grid_layout = MosGrid(
        l=10 * lmin, w=20 * lmin, m=5, model="pch", rows=4, add_guardring=True
    )
    d >> grid_layout % g >> s
    place(grid_layout)

5. 辅助方法与属性

Mos (及其子类 NMOS, PMOS) 实例提供了许多有用的方法和属性：

  • 参数访问 (attr.py): .l(), .w(), .m(), .nf(), .model(), .from_virtuoso(), .gate(), .drain(), .source(), .body() 等。
  • 版图几何信息 (attr.py): .drain_aa_spacing(), .source_gate_spacing(), .drain_width(), .gate_extrude_height(), .gate_spacing(), .mosfet_overlap_size() 等（需要先 place()）。
  • 端口访问 (method.py):
    • .get_left/right_ds(): 获取最左/右的 D/S 端口 NRect。
    • .get_left/right_rect_node(): 获取最左/右的 D/S 节点。
    • .get_left/right_gate/drain/source(): 获取指定端口的最左/右 NRect。
    • .get_ds(): 获取所有 D/S 端口 NRect 的列表。
  • 状态检查 (method.py): .drain/source_on_left/right(), .is_node_shared(other_mos)。
  • 操作 (method.py):
    • .singularize_mos(): 将 m>1 的 MOS 拆分为 m 个 m=1 的 MOS 列表。
    • .to_poly_dummy(): 将 MOS 转换为 Poly Dummy 矩形列表（会 kill 自身）。
  • 仿真/模型 (model.py, sim.py):
    • .get_ids(vgs, vds, vbs): 获取指定偏置下的 Id。
    • .get_gm(vgs, vds, vbs): 获取指定偏置下的 gm。
    • .sim(vgs, vds, vbs): 运行 DC 扫描并返回包含工作点参数的字典。

6. 详细使用样例：差分对与电流镜负载

这个例子结合了 PairedMosfet（差分对）、CurrentMirror（有源负载）和基本 NMOS（尾电流源），并使用布局助手进行组装。

from pyted import *
from ted_device import NMOS, PMOS, PairedMosfet, CurrentMirror
from layout import guardring, row, col, place
from algo import route, layout_mos
from ted_builtins import VSource, ISource # 用于 Testbench

@module
def DiffAmp(vinp: Input, vinn: Input, voutp: Output, voutn: Output, ibias: Input):
    # --- 定义内部节点 ---
    tail_node = Node("tail")
    pm_gate = Node("pm_gate") # 电流镜栅极偏置

    # --- 参数 ---
    diff_l, diff_w, diff_m = 1e-6, 8e-6, 4
    load_l, load_w, load_m = 0.5e-6, 4e-6, 2
    tail_l, tail_w, tail_m = 2e-6, 20e-6, 8

    # --- 1. 差分输入对 (NMOS, Cross-Coupled) ---
    nmos_pair = PairedMosfet(
        l=diff_l, w=diff_w, m=diff_m, model="nch",
        mode="cross", dummy_m=1 # 加 dummy 改善匹配
    )
    # 连接: (D1, D2) >> pair % (G1, G2) >> S
    (voutn, voutp) >> nmos_pair % (vinp, vinn) >> tail_node

    # --- 2. 电流镜负载 (PMOS, Type1) ---
    pmos_load = CurrentMirror(
        l=load_l, w=load_w, m=load_m, model="pch", type="type1",
        place_mode=HORIZONTAL # 水平放置以便与差分对堆叠
    )
    pmos_load.input(voutn, mid=pm_gate) # 输出端接二极管连接作为输入
    pmos_load.output(voutp)             # 输出端连接到差分对另一输出

    # --- 3. 尾电流源 (NMOS) ---
    tail_current_source = NMOS(l=tail_l, w=tail_w, m=tail_m, model="nch")
    tail_node >> tail_current_source % ibias >> Vss()

    # --- 4. 布局 ---
    # 使用 layout_mos 对各个部分进行初步布局
    nmos_pair_layout = layout_mos(nmos_pair) # PairedMosfet 内部已处理布局
    pmos_load_layout = pmos_load.layout()      # CurrentMirror 提供 layout 方法
    tail_current_layout = layout_mos(tail_current_source)

    # 添加保护环
    nmos_pair_guarded = guardring(nmos_pair_layout, Vss())
    pmos_load_guarded = guardring(pmos_load_layout, Vdd())
    tail_current_guarded = guardring(tail_current_layout, Vss())

    # 整体排列
    final_layout = col(
        pmos_load_guarded,
        nmos_pair_guarded,
        tail_current_guarded,
        spacing=1000, # 各部分间距
        align=CENTER
    )

    place(final_layout)
    route(final_layout)

这个详细样例展示了：

  1. 使用 PairedMosfet 创建差分对。
  2. 使用 CurrentMirror 创建有源负载。
  3. 使用 NMOS 创建尾电流源。
  4. 使用 layout_mos 和 guardring 进行布局。
  5. 使用 col 组合最终版图。
  6. 在 Testbench 中设置输入激励并进行仿真/验证的框架。
