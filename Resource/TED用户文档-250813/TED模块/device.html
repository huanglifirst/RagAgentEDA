TED 器件库 (ted_device)

1. 概述

ted_device 模块是 TED 中用于描述和生成常见模拟和数字电路基本单元（器件）的核心库。它基于 ted_builtins 提供的底层仿真和 LVS 模型，并通常增加了以下功能：

  • 集成视图: 将器件的仿真模型 (type="sim" builtin)、LVS 模型 (type="lvs" builtin) 和版图生成逻辑 (type="layout" moduledef) 封装在同一个 Python 模块中（例如 NMOS, RpolySingle, MomCap）。
  • 参数化版图生成 (PCell 功能): 许多 ted_device 组件是参数化的单元生成器。用户只需提供关键参数（如 L, W, M, 模型名称），模块内部会自动计算并生成符合设计规则的版图。
  • PDK 适应性: 通过内部的参数映射 (translate.py) 和模型查找机制，尝试适配不同的 PDK。
  • 常用模拟结构: 提供了对常用模拟电路结构（如电流镜、差分对）的封装和版图生成支持。
  • 高级布局模式: 支持如共质心 (Common-Centroid)、ABBA、交叉耦合 (Cross-Coupled) 等高级版图布局技术。
  • 辅助工具: 包含用于器件布局（如 layout_mos, row_mos）、保护环 (guardring)、仿真激励 (stimuli) 等的辅助函数和类。

本篇文档将分类介绍 ted_device 中的主要组件、高级结构、仿真激励以及相关工具。

2. 核心器件模块 (components)

ted_device.components 子目录包含了各种基本器件的定义。这些模块通常都封装了对应的 LVS、仿真和版图视图。

2.1 MOSFET (components/mos)

提供了 NMOS 和 PMOS 晶体管的生成器。

  • Mos(l, w, m, nf, model, from_virtuoso, obs, builtin, **params): 基础 MOS 类（通常不直接使用）。
  • NMOS(g, d, s, l, w, nf, m, model, from_virtuoso, obs, builtin, **params): N 型 MOSFET。
  • PMOS(g, d, s, l, w, nf, m, model, from_virtuoso, obs, builtin, **params): P 型 MOSFET。
  • 主要参数:
    • g, d, s: 栅、漏、源节点 (Ctrl, Input, Output)。体端 (b) 通常根据 model 自动连接到 Vss() 或 Vdd()。
    • l, w: 沟道长度和宽度 (单位: 米)。若不指定，会使用 PDK 中定义的默认值。
    • m, nf: 乘数或 finger 数量 (两者不能同时指定，m 优先)。
    • model: PDK 中的 MOS 模型名称 (如 "nch", "pch_lvt")。
    • from_virtuoso: (bool) 是否尝试从 Virtuoso 获取版图和模型信息 (通过 setenv 设置全局默认值)。True 时调用 MosFromVirtuosoLayout，False 时调用 MosLayoutArray。
    • obs: (Layer 或 List[Layer]) 指定布线障碍层。
    • builtin: (bool) 是否生成对应的 Sim/LVS builtin 实例 (默认为 True)。
  • 内部组件:
    • MosLayout: 单个 (m=1) MOS 管的版图生成器。
    • MosLayoutArray: 多 finger MOS 管 (m>1) 的版图生成器，通过排列 MosLayout 实现。
    • MosFromVirtuosoLayout: 从 Virtuoso 导出的 GDS 生成版图。
    • Mosfet, MosfetLvs, MosfetFromVirtuoso*: 来自 ted_builtins 的 Sim/LVS 模型。
  • 特殊版本:
    • PMOS5T, NMOS6T: 5 端和 6 端 MOSFET (如 HV 或 DNW)，包含额外的衬底/阱连接。

示例 (基本实例化)

from ted_device import NMOS, PMOS, layout_mos, guardring

@module
def InverterLayout(vin: Input, vout: Output):
    # 实例化 PMOS 和 NMOS
    pm = PMOS(l=180e-9, w=1e-6, m=2)
    nm = NMOS(l=180e-9, w=0.5e-6, m=2)

    # 连接成反相器
    Vdd() >> pm % vin >> vout
    vout >> nm % vin >> Vss()

    # 使用 layout_mos 进行布局 (会自动处理源漏共享)
    pm_layout = layout_mos(pm)
    nm_layout = layout_mos(nm)

    # 添加保护环并整体布局
    final_layout = col(
        guardring(pm_layout, Vdd()),
        guardring(nm_layout, Vss()),
        spacing=500 # 保护环间距
    )
    place(final_layout)
    route(final_layout) # 对单元间连接进行布线

2.2 电阻 (components/res)

主要提供 Poly 电阻的生成器。

  • RpolySingle(a, b, l, w, r, model, contact, obs, expanded_size, builtin): 单个 Poly 电阻单元。
    • 封装了 RpolyLayout (版图) 和 RpolyBuiltin (Sim/LVS)。
    • l, w: 电阻 Poly 层尺寸。
    • contact: (int) 控制 Contact 孔数量/排列方式。
    • obs: 障碍层。
    • expanded_size: 在外围添加 TEXT 层边界的大小。
    • builtin: 是否包含 LVS/Sim 模型。
  • Rpoly(a, b, l, w, r, m, model, connection, contact, spacing, obs, ...): 用于创建串联或并联的 RpolySingle 阵列。
    • m: 单元数量。
    • connection: SERIES 或 PARALLEL。
    • 内部调用 SeriesResistor 或 ParallelResistor。
  • SeriesResistor(res, l, w, model, ...): 高阶函数，接收一个基础电阻模块 (res，如 RpolySingle)，返回一个可以生成 n 个 res 串联阵列的新模块。
  • ParallelResistor(res, l, w, model, ...): 类似 SeriesResistor，但生成并联阵列。
  • Res(a, b, r, m, mode): 使用 ted_builtins.Resistor 创建简单的串联 (mode="serial") 或并联 (mode="shunt") 理想电阻网络。

示例 (创建串联电阻阵列)

from ted_device.components.res import RpolySingle, SeriesResistor

@module
def ResistorChain(inp: Input, outp: Output):
    # 创建一个 RpolySingle 的 SeriesResistor 生成器
    # 每个单元 l=10u, w=1u, model='rphrpo'
    series_rpoly = SeriesResistor(RpolySingle, l=10e-6, w=1e-6, model="rphrpo")

    # 使用生成器创建 5 个串联的 RpolySingle 实例
    # dummy_m=1 表示在两端各加一个 dummy 电阻
    # row_num=2 表示布局时折叠成 2 行
    res_chain_inst = series_rpoly(inp, outp, n=5, dummy_m=1, row_num=2)

    # (可选) 添加保护环等
    # guardring(res_chain_inst, Vss())
    place(res_chain_inst)
    # route() # 通常 series_layout 内部会进行布线

2.3 电容 (components/cap)

提供多种电容器件生成器。

  • MomCap(..., model, ...): Metal-Oxide-Metal 电容。
    • 如果 model 提供（如 crtmom, cfmom），则调用 MomcapFromVirtuoso 从 Virtuoso GDS 生成版图，并使用对应的 LVS/Sim 模型。
    • 如果 model 为 None，则调用 MomCapTed 使用 TED 内建逻辑生成版图。
    • 参数包括 finger_* (尺寸, 间距, 数量), bottom/top_layer, well_type, c (目标电容值, 可用于反推尺寸或仅用于仿真)。
  • MimCap(...): Metal-Insulator-Metal 电容。调用 PDK 特定的实现（如 MimCapForT65, MimCapForS180bcd）。
  • SarCapUnit(CB, CT, c, ...): SAR ADC 中常用的单位电容。包含一个从 GDS 硬编码的版图 (OriginalSarCapUnit) 和一个用于仿真的 ted_builtins.C。
  • SARCapGrid(s=[...], ...): 根据输入的权重列表 s，使用 SarCapUnit 构建共质心布局的电容阵列。
  • MosCap(a, b, l, w, m, ...): MOS 电容。使用 Mos 组件实现版图和模型。
  • Decap(w, h, rows, cols, ...): 去耦电容阵列。使用 PMOS 作为单元构建版图。

示例 (创建 MOM 电容)

from ted_device.components.cap import MomCap
from tedcore import M3, M5

@testbench
def MomCapExample():
    n1, n2 = Node(2)

    mom1 = MomCap(
        n1, n2,
        finger_l=15e-6,
        finger_w=0.15e-6,
        finger_spacing=0.15e-6,
        finger_num=40,
        bottom_layer=M3,
        top_layer=M5,
        c=2e-12 # 提供电容值用于仿真模型
    )

    # 创建一个使用 PDK 模型 (Virtuoso Pcell) 的 MOM 电容
    # mom2 = MomCap(n1, n2, model="crtmom", num_h=32, num_v=32) # 参数需匹配PDK

    place(mom1)
    # mom1.lvs()

2.4 BJT (components/bjt)

  • BJT(C, B, E, model): 双极结型晶体管 (Bipolar Junction Transistor) 的主入口。
  • NPN(...), PNP(...): 类型特定的封装器。
  • 参数: C, B, E (节点), model (PDK 模型名，如 "npn5", "pnp10")。
  • 行为:
    • get_bjt_cellname_and_model: 从 model 推断 Virtuoso cellname 和 实际 model。
    • get_bjt_size: 从 model 推断尺寸参数 (e.g., "npn5" -> size 5)。
    • 调用 BjtLayout (从 Virtuoso GDS), BjtLvs, BjtSim (或 4 端版本) 来创建不同视图。

示例:

from ted_device.components.bjt import NPN, PNP

c, b, e = Node(3)
# 实例化一个 NPN BJT，使用模型 "npn10"
npn_inst = NPN(c, b, e, model="npn10")
# 实例化一个 PNP BJT
pnp_inst = PNP(c, b, e, model="pnp5")

  • Diode(plus, minus, l, w, model): 主二极管模块。

  • 参数: plus, minus (节点), l, w (尺寸), model (PDK 模型名, 如 "ndio", "pdio")。

6. 高级版图结构

ted_device 提供了一些用于生成常见模拟版图结构的模块或类。

6.1 对称匹配对管 (PairedMosfet) (components/mos/paired_mos)

  • 功能: 生成用于差分对等需要匹配的场景的 MOSFET 对。
  • 签名: PairedMosfet(d1, d2, g1, g2, s, l, w, m, nf, mode, model, dummy_l, dummy_m, ...)
  • 参数:
    • d1, d2, g1, g2, s: 差分对的漏极、栅极和公共源极。
    • mode: 版图模式：
      • "abba": ABBA 排列。
      • "cross": 交叉耦合排列 (使用 M1/M2 交叉)。
      • "cross_high_layer": 交叉耦合排列 (使用更高层金属交叉)。
    • dummy_m: 在外围添加的 Dummy 管数量 (每侧)。
    • dummy_l: Dummy 管的长度。
  • 内部: 调用 ABBA, Cross, 或 CrossHighLayer 模块生成版图。同时实例化对应的 Mosfet/MosfetLvs 用于 LVS/Sim。
  • 特殊版本: PairedNMosfet5T, PairedNMosfet6T 继承并添加了 5/6 端子连接。

示例 (改编自 test_paired_mos.py::test01)

@module

6.2 电流镜 (CurrentMirror) (cm/current_mirror.py)

  • 功能: 生成各种类型的电流镜结构，包括简单的、Cascode、带电阻的等。
  • 签名: CurrentMirror(l, w, m, model, type, place_mode, l_dummy, dummy_m, res, ds, dummy, from_virtuoso)
  • 参数:
    • l, w, m, model, from_virtuoso: 基本 MOSFET 参数。l, w, m 可以是列表（用于 Cascode 等）。
    • type: 电流镜类型 ("type1" 到 "type5"，对应不同电路结构)。
    • place_mode: 布局模式 (HORIZONTAL, VERTICAL, EULER)。
    • l_dummy, dummy_m: Dummy 管参数。
    • res: (用于 type3) 提供一个电阻实例 (如 RpolySingle())。
    • ds: layout_mos 的 ds 参数。
    • dummy: 是否自动添加 dummy (用于 place_mode=VERTICAL)。
  • 方法:
    • .input(Idc, VB=None, PD=None, mid=None, ...): 定义输入/参考支路连接。
    • .output(Iout, k=1, m=None, single=False, mid=None, ...): 定义输出支路。k 是电流比例，m 可直接指定输出管尺寸，single 用于 Cascode 的单端输出。
    • .layout() 或 .col(), .row(), .get_row(): 获取生成的版图 InstList。
    • .rpoly: (用于 type3) 获取内部使用的电阻实例。
  • 行为: 内部实例化多个 NMOS/PMOS，根据 type 连接它们，并根据 place_mode 调用 layout_mos 或 autolayout_euler 生成版图。

示例 (改编自 cm2_layout_test.py::Cascode2)

from ted_device.components.res import RpolySingle
from ted_device import CurrentMirror, guardring

@module
def CascodeMirrorExample(Idc: Input, Iout1: Output, Iout2: Output):
    l = [2e-6, 2e-6] # [M1_L, M2_L] for Cascode
    w = 1e-6
    m = 2 # 每个 MOS M=2

    # 创建一个 Type3 (带源极电阻) 的 Cascode 电流镜
    cm = CurrentMirror(
        l=l, w=w, m=m, model="nch", type="type3",
        res=RpolySingle(l=5e-6, w=0.5e-6), # 提供源极电阻实例
        place_mode=HORIZONTAL,
        l_dummy=get_mosfet_min_l("nch"),
        dummy_m=1 # 添加 dummy
    )
    cm.input(Idc) # 连接输入电流
    cm.output(Iout1, k=1) # 输出 1:1
    cm.output(Iout2, k=2) # 输出 2:1 (m=4)

    # 布局：将电阻放在旁边，电流镜主体加保护环
    layout = row(cm.rpoly, guardring(cm.layout(), Vss()), spacing=500)
    place(layout)
    route(layout)

6.3 其他阵列和辅助 (resnet.py, mos_grid.py, dummy.py)

  • SeriesResistor, ParallelResistor: 如 2.2 节所述，用于创建电阻阵列。

7. 仿真激励 (stimuli)

提供常用的仿真信号源。

  • Clock(out, val0, val1, period, delay, rise, fall, width): 生成脉冲电压源 (VSource(mode="pulse"))。
  • Vdc(a, b, dc=1): 生成直流电压源 (VSource(dc=...))。
  • Vsin(a, b, dc, amp, freq, ac, delay): 生成正弦电压源 (VSource(mode="sine"))。
  • Vnoise(a, b, amp, fmax, n): 生成简单的噪声源（多个随机正弦波叠加）。

8. 实用工具和属性

  • Parameter Translation (translate.py): translate_mosfet_params, translate_resistor_params 等函数，用于将 TED 标准参数（如 l, w）转换为特定 PDK 的 Virtuoso PCell 参数名和格式。
  • MOS Helpers (mos_algorithm.py, method.py, attr.py, fields.py):
    • get_mosfet_*_l/w: 获取模型默认或最小/最大尺寸。
    • check_mosfet: 检查 L/W 是否在允许范围内。
    • row_mos: 自动排列 MOS 管并处理源漏共享。
    • mos_outer_ring, guardring: 添加保护环。
    • MosFields: 获取 MOS 版图几何参数。
    • Mos.get_ids(), Mos.get_gm(): (需要 gmid 库) 获取器件工作点参数。
    • Mos.singularize_mos(): 将 m > 1 的 MOS 拆分为多个 m=1 的 MOS。
    • Mos.get_left/right_ds(): 获取最左/最右的源漏端口。
  • Empty Placeholder (empty.py): Empty() 模块可用于在布局中占位，在最终 place() 时会被移除。filter_empty() 可用于从列表或 InstList 中移除 Empty 实例。

9. 详细使用样例：带 Dummy 的电流镜布局与验证

from pyted import *
from ted_device import *
from layout import guardring, row, col, place
from algo import route

@module
def AdvancedCurrentMirror(idc_in: Input, iout: Output, ratio: float = 2.0):
    # --- 器件参数 ---
    mos_l = 0.5e-6
    mos_w = 5e-6
    input_m = 4      # 输入管 M 值
    dummy_count = 2  # 每侧 Dummy 数量
    model = "nch"   # 使用 NCH 模型

    # --- 1. 创建电流镜实例 (不立即生成版图或 builtin) ---
    #    注意：类型为 type1， place_mode=HORIZONTAL
    #    dummy=True 使得可以使用 dummy_m 参数
    #    builtin=False 避免重复生成 builtin
    cm = CurrentMirror(
        l=mos_l, w=mos_w, m=input_m, model=model, type="type1",
        place_mode=HORIZONTAL, # 使用水平排列（易于添加竖直 dummy）
        l_dummy=mos_l,       # Dummy L 与主器件相同
        dummy_m=dummy_count, # 每侧加 2 个 dummy
        dummy=True,
        from_virtuoso=False, # 使用 TED 生成版图
        builtin=False        # 稍后手动添加 LVS/Sim 模型
    )

    # --- 2. 定义输入和输出支路 ---
    cm.input(idc_in)
    cm.output(iout, k=ratio) # 输出电流为输入的 ratio 倍

    # --- 3. 生成带 Dummy 的版图 ---
    #    layout() 方法会根据 place_mode (HORIZONTAL) 调用相应的布局函数
    cm_layout_instlist: InstList = cm.layout()

    # --- 4. 添加 LVS 和 Sim 模型 ---
    #    手动为输入管和输出管添加 builtin 模型
    #    输入管
    inp_mos = cm.get_input_mos(0) # 获取输入管实例列表 (type1 只有一行)
    (inp_mos.drain(), inp_mos.gate(), inp_mos.source(), Vss()) >> MosfetLvs(l=mos_l, w=mos_w, m=input_m, model=model)
    (inp_mos.drain(), inp_mos.gate(), inp_mos.source(), Vss()) >> Mosfet(l=mos_l, w=mos_w, m=input_m, model=model)
    #    输出管
    out_mos_list = cm.get_output_mos(0, 0) # 获取第一个输出支路的所有 MOS 实例
    out_m = cm.compute_scaled_m(ratio) # 计算输出管的 M 值
    (out_mos_list[0].drain(), out_mos_list[0].gate(), out_mos_list[0].source(), Vss()) >> MosfetLvs(l=mos_l, w=mos_w, m=out_m, model=model)
    (out_mos_list[0].drain(), out_mos_list[0].gate(), out_mos_list[0].source(), Vss()) >> Mosfet(l=mos_l, w=mos_w, m=out_m, model=model)
    #    Dummy 管 (如果需要精确 LVS/仿真模型，也需添加，但通常不需要)
    #    ...

    # --- 5. 添加保护环 ---
    guarded_cm = guardring(cm_layout_instlist, connectTo=Vss(), vias=2)

    # --- 6. 最终放置与布线 ---
    place(guarded_cm)
    route(guarded_cm)

@testbench
def TopAdvancedCM():
    idc_node, iout_node = Node(2)
    # ... 添加 VSource/ISource 等激励 ...
    dut_inst = AdvancedCurrentMirror(idc_node, iout_node, ratio=1.5)

    # dut_inst.lvs() # 执行 LVS
    # dut_inst.drc() # 执行 DRC
    dut_inst.to_gds("advanced_cm.gds")

这个详细样例展示了：

  1. 使用 CurrentMirror 类来配置一个带 Dummy 的电流镜。
  2. 通过 input() 和 output() 方法定义电路连接。
  3. 调用 .layout() 方法生成包含 Dummy 的水平布局 (place_mode=HORIZONTAL)。
  4. 手动实例化 Mosfet 和 MosfetLvs 来定义 LVS 和仿真网表（因为 builtin=False）。
  5. 使用 guardring 添加保护环。
  6. 最后进行 place 和 route。
